# Customer Segmentation Analysis (R)

## Descripción
Proyecto de análisis de datos enfocado en comprender el comportamiento de clientes de una empresa retail.

## Objetivo
Identificar patrones de consumo, segmentar clientes y generar insights para la toma de decisiones comerciales.

## Herramientas
- R
- ggplot2
- dplyr

## Análisis realizados
- Limpieza de datos
- Feature engineering
- Segmentación tipo RFM
- Análisis exploratorio (EDA)
- Detección de outliers
- Visualización de datos

## Resultados
- Identificación de clientes de alto valor
- Relación entre ingreso y gasto
- Segmentación para estrategias comerciales
